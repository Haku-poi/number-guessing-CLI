# How to play
1- Difficulty level: There's 3 difficulty level you can choose in total, they are:

    1. Easy (10 chance)
    2. Medium (5 chance)
    3. Hard (3 chance)
2- The computer will generate a random number between 1 and 50.

3- If you guess the same number as the computer, you win.

4- To leave, all you need to do is choose "4" on the choices.

Have fun playing the game!
